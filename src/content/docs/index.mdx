**Every part of your life — wealth, health, tax, career — is a folder of markdown.** Open one, ask a question, and Prevail fans it out to every AI CLI you already have logged in. A chair model reads all the replies and writes one decisive answer, with a dedicated panel surfacing the _disagreement_ — which is the point.

Works from your terminal or your phone. Single binary. No daemon, no Docker, no API keys to manage.

```bash
curl -fsSL https://prevail.sh/install | bash
prevail demo
```

## Why Prevail

<CardGrid>
  <Card title="A council, not a single voice" icon="comment-alt">
    Four models answer in parallel. One chair synthesizes. Disagreement gets its own panel — that's where the value is.
  </Card>
  <Card title="Domain = folder" icon="open-book">
    `wealth/`, `health/`, `tax/`. Plain markdown. Edit anywhere. Sync with git, iCloud, or Tailscale. No database.
  </Card>
  <Card title="Uses CLIs you already pay for" icon="seti:powershell">
    Spawns `claude`, `codex`, `agy`, `ollama` — inherits every login, MCP server, and skill. No API keys to wire up.
  </Card>
  <Card title="Local-private when it matters" icon="approve-check">
    Ollama auto-detected at `localhost:11434`. Run health or wealth on a council that never leaves your machine.
  </Card>
  <Card title="Off the keyboard" icon="phone">
    A Telegram bridge puts the same cockpit, the same council, on your phone — with a mandatory chat-ID allowlist.
  </Card>
  <Card title="Hardened by default" icon="shield">
    Env-scrubbed subprocesses, prompt-injection–aware operating manual, OAuth tokens at `chmod 0600`.
  </Card>
</CardGrid>
